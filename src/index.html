使用帮助

1. 数据管理

支持上传 TIF、TIFF、SHP 格式的文件。TIF/TIFF 为多光谱影像文件，SHP 为地块矢量文件。上传后可在页面下方预览文件信息。

2. 植被特征分析

选择已上传的TIF文件，根据勾选顺序为波段排序，并指定每个序号对应的波段类型（蓝光、绿光、红光、近红外、红边），系统将自动计算 NDVI、RVI 等植被指数。

3. 处方图生成

选择TIF和SHP文件，设置地块参数，系统将生成施肥处方图，支持导出为 TIF、PNG 或 CSV 格式。

4. 大疆智农适配

将处方图导出为 TIF+TFW 格式，可直接导入大疆智农 App 使用。

数据格式建议

TIF文件应为多光谱正射影像，包含多个波段数据。SHP文件应包含地块边界信息。坐标系建议使用 WGS84 或 CGCS2000。
